## 1. Introduction

**Knowledge Reset** is the global knowledge repository that serves as the central brain for the entire Digital Reset ecosystem. Its purpose is to hold all documentation, knowledge bases, instructions, manuals, project documentation, customizations, and internal knowledge generated by both Digital Reset and its customers.

This repository contains the complete documentation for the Knowledge Reset platform, including its architecture, product requirements, standard operating procedures, and technical specifications.

## 3. Key Information

*   **Technology Stack:** Python, GraphQL, Supabase (PostgreSQL with pgvector), Vercel, Google Antigravity
*   **AI APIs:** Claude Opus 4.5, Gemini 3 Pro, GPT-5.2
*   **Core Capabilities:** Centralized knowledge consolidation, automated content acquisition, conversational AI interface, intelligent retrieval, and robust APIs for integration.
